---
name: 音乐文件搜索
description: 当用户要求搜索、查找、列出本机上的音乐文件、歌曲、音频文件时触发
---

当用户请求搜索音乐或音频文件时，执行以下步骤：

1. 使用 bash 工具执行以下命令搜索常见音频格式：
   `find ~/ -type f \( -name "*.mp3" -o -name "*.wav" -o -name "*.flac" -o -name "*.ogg" -o -name "*.aac" \) 2>/dev/null`

2. 如果搜索结果过多（超过 50 个文件），只展示前 20 个并告知用户文件较多。

3. 如果未找到任何音频文件，告知用户当前系统中没有音乐文件。

4. 找到文件后，向用户汇报文件数量和文件列表（含路径），并询问是否需要播放。
